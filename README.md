# MRI Tumor Detector 🧠

This project detects brain tumors from MRI images using Deep Learning (CNN).

## Features
- Tumor detection from MRI scans
- Deep Learning model (CNN)
- Image classification: Tumor / No Tumor

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

## Dataset
Brain MRI dataset used for training.

## How to run
1. Clone repo
2. Install requirements
3. Train model
4. Run prediction script
